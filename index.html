A to Extensify

Listed below are some of our favourite projects from extensify:

  • 1. OCR Psuedocode

    In the UK many students go through the struggles of learning syntax for examboard-designed Pseudocode, this extension helps said students during their coursework by providing autoformatting to the specified Pseudocode syntax.

  • 2. Choo Choo Code

    This extension uses real-time GTFS data to determine what train you're currently travelling on in order to subsequently show you relevant information about the train's journey. However, at time of writing this extension only support German Deutsche-Bahn Trains.

  • 3. Stats Tracker

    This extension tracks your coding statistics, like number of saves, average line length, and more, and displays it in a useful small widget.

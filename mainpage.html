What is Extensify?

Extensify is a Hack Club You Ship We Ship Program (YSWS) where we'll ship you a special keychain or a premium VS Code Theme, whenever you ship us an IDE Extension.

We want to encourage you to build and ship your own IDE extensions, whether it's for VS Code, JetBrains, or any other IDE. We can't wait to see what you create and we encourage you to push it out into the world!

IDE Extensions are power tools, they can help you automate tasks, improve your workflow, and make your coding experience more enjoyable. Whether it's adding a virtual pet to run around your code, or something that helps you with your code formatting, they make your editor yours, and once you start using them you'll never be able to turn back!

Extension Examples

Listed below are some of our favourite extension examples, for if you're looking for inspiration, or just for your next customisation:

  • 1. VS Code Pets

    Adds a virtual pet to your editor that you can interact with while coding. - It's so much fun to just glance down and see it running around, plus you can throw balls for them to catch!

  • 2. Prettier - Code formatter

    Automatically formats your code for consistency and readability, wrapping it perfectly so it never flows off your screen.

  • 3. Github Copilot

    AI-powered code completion and suggestions as you type. A very popular vibe-coding tool, that's codecompletion feature is incredibly useful.

  • 4. Github Pull Requests

    Manage GitHub pull requests and issues directly from VS Code, it gives a mini Github GUI in your IDE.

  • 5. In Your Face

    An extension that shows you how many errors your code has in real-time, through displaying progressively more memed pictures of Mr. Incredible.

  • 6. Live Preview

    Lets you launch a local development server with live reload for static and dynamic pages.

  • 7. VS Code PDF Integration

    Allows you to view and annotate PDF files directly within VS Code.

  • 8. VS Code Drawio

    Lets you create and edit diagrams using Draw.io inside your editor. - So helpful when creating documentation.
